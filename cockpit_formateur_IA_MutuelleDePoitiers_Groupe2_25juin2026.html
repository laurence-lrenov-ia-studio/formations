  • Demander « vous avez compris ? » — vérifier par la pratique.
  • Laisser une question hors-sujet dériver plus de 2 minutes.

08 Clôture — checklist de fin

☁ Cloud — enregistrement live

Branche le cockpit à ton Google Sheet : chaque sauvegarde part aussi dans le cloud (écriture immédiate + journal). Le local reste ton filet hors-ligne ; le cloud, c'est ta trace qui dure et te suit d'un PC à l'autre.

Connexion

Cloud : non configuré (le cockpit fonctionne en local).

Comment ça marche

  • Auto-push : dès que l'URL est renseignée, chaque sauvegarde locale est aussi envoyée au Sheet (anti-rafale : au plus une fois toutes les ~3 s).
  • Hors-ligne : si le wifi tombe, le local continue ; le cloud rattrape à la sauvegarde suivante.
  • Charger depuis le cloud : récupère la dernière version de CETTE session. Ça remplace l'état à l'écran.
  • Rien ne se perd : le Sheet garde un onglet « log » où chaque envoi est ajouté, jamais écrasé.
